# 데이터셋 수치 변경 이력 (Dataset History)

이 문서는 프로젝트 진행 중 Notion에 기록된 "전체 영상 개수"가 시점마다 다르게 보고된 이유를 추적하기 위해 작성되었습니다. **팀 내부적으로 208개와 215개라는 서로 다른 숫자가 통용되었을 가능성이 제기되었으나, 이번 조사(Notion 회의록 20건 + 기술 문서 13건 전수 확인)에서 "215개"라는 숫자는 어떤 페이지에서도 발견되지 않았습니다.** 아래는 실제로 Notion에서 확인된 숫자들과 그 출처입니다.

|---|---|---|---|

## 판단 및 근거

1. **가장 최신이자 가장 상세한 수치는 208개(0611)** 입니다. 종목별 세부 내역과 프레임 단위 검출 통계까지 함께 기록되어 있어 다른 수치보다 신뢰도가 높다고 판단했습니다.
2. 0428의 209개(학습 split 기준)와 0611의 208개를 종목별로 비교하면 `squat`(89)와 `benchpress`(94)는 완전히 일치하지만, **`deadlift`만 26개(0428) → 25개(0611)로 1개 차이**가 있습니다. 0611 회의록에는 "데드리프트 개인 크롤링 6개 추가"라는 기록도 있어, 4~6월 사이 데드리프트 영상이 추가/삭제/교체되었을 가능성이 있습니다. 정확한 원인(중복 제거, 라벨 오류로 인한 삭제, 재촬영 등)은 이번 Notion 조사만으로는 특정할 수 없었습니다.
3. **215개는 이번 조사 범위(Notion 워크스페이스 검색 + 관련 하위페이지 전수 확인) 내에서 확인되지 않았습니다.** GitHub 저장소에도 실제 CSV/데이터 파일이 없어 대조가 불가능했습니다. 팀 내부의 별도 스프레드시트나 구두 논의에서 나온 숫자일 가능성이 있으나, 이 문서만으로는 근거를 확인할 수 없으므로 "미확인"으로 남겨둡니다.
4. 652개, 134개는 각각 파일럿 실험용/부분 실험용 데이터셋이므로 "3대 운동 전체 데이터셋 크기"와 혼동하지 않도록 이 문서에 구분해 기록했습니다.

## 현재 README/dataset.md에 반영하는 수치

**208개(squat 89 / benchpress 94 / deadlift 25)** 를 1학기 종료 시점 기준 최신 수치로 채택합니다. 단, 이 수치도 팀이 보유한 실제 최신 CSV/라벨 파일을 직접 집계한 것이 아니라 Notion 기록을 근거로 한 것이므로, **2학기 시작 시 실제 데이터 폴더/CSV를 기준으로 재집계하여 이 문서를 갱신할 것을 권장합니다.**

## 추가로 확인이 필요한 항목

- deadlift 26 → 25 변경의 정확한 원인 (삭제된 영상이 무엇인지, 중복이었는지)
- "약 200개"라는 0611 서술과 정확한 표(208개) 사이의 반올림 표현 차이는 실질적 불일치가 아닌 것으로 판단했으나, 팀 확인 권장
- 215개 수치의 실제 출처 (해당 자료가 존재한다면 Notion에 추가 기록 후 이 문서를 업데이트)
